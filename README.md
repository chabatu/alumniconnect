# AlumniConnect

AlumniConnect is a LinkedIn-like platform designed specifically for university communities, bringing together current students, alumni, and professionals from leading companies.

The platform enables graduates and students of a university to connect, share experiences, and create meaningful professional relationships. Alumni working in major companies can post job opportunities, internships, and career advice directly to students and fellow graduates.

## Features

- 🎓 Connect current students with alumni
- 💼 Job and internship postings from verified alumni
- 🤝 Networking within university communities
- 📢 Share career insights and experiences
- 🔍 Explore opportunities from top companies
- 🏫 University-based community structure

## Purpose

AlumniConnect aims to bridge the gap between education and industry by creating a trusted network where students can access real-world opportunities through alumni connections.

## Tech Stack

- Frontend: HTML / CSS
- Backend: Pyhton / Java
- Database: PostgreSQL / MongoDB
